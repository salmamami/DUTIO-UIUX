# DUTIO

UI/UX Design aplikasi manajemen tugas mahasiswa.

## Description
DUTIO adalah rancangan aplikasi untuk membantu mahasiswa mengatur tugas, jadwal, dan aktivitas akademik secara lebih terstruktur.

## Design Process
Project ini dibuat melalui beberapa tahapan:
- Low Fidelity Wireframe (User & Koordinator)
- High Fidelity Prototype (User & Koordinator)
- User Flow Design
- Dashboard Planning

## Roles
### User
- Melihat tugas
- Melaksanakan pembagian tugas 
- Melihat reminder
- Upload laporan berupa foto area piket

### Koordinator
- Mengelola tugas anggota
- Monitoring progress
- Mengatur jadwal kelompok
- Menilai serta mengevaluasi hasil tugas berdasarkan laporan
